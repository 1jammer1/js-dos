DOS Sokol

DOS Sokol uses the same codebase as worker and direct backends. This is native version of emulator backend, it can be compiled to run on PC. You can compile it with make:

Run
make sokol

Sokol version is used to develop DOS emulator backend. Read more how to develop it in the Contributing guide.
