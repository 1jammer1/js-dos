Overview

DOS.Zone - is a js-dos community project. The main target of it is to build repository of js-dos bundles, that can be easily used to integrate DOS programs on any website.

DOS.Zone consist of:

  • discord to instant talks.
  • talk.dos.zone is a place to discuss and publish new js-dos bundles. Approved bundles are pushed in dos.zone repository.
  • dos.zone it's a default player for js-dos bundles based on js-dos api.
  • game studio it's a full featured bundles creator for js-dos.
  • twitter our news channel

Database#

Our database if free for use, you can obtain it here. It's synced with Talks in both directions. You can find all links to bundles in it.
